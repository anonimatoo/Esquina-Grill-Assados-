Esquina Grill Assados - App Digital
Descricao
Este projeto e uma aplicacao web mobile-first desenvolvida para o Esquina Grill Assados. Ele funciona como um cartao de visita digital interativo com uma introducao cinematografica, efeitos de fogo realista, luzes neon e redirecionamento automatico.
Funcionalidades
 * Intro Cinematografica: Animacao de logotipo com efeito de respiracao e fogo intenso.
 * Efeito de Brasas: Particulas de fogo flutuantes criadas via JavaScript (Canvas API).
 * Timer de Loop: A introducao dura exatamente 4 segundos e transita suavemente para o cartao principal.
 * Design Neon e LED: Interface escura com bordas brilhantes e efeitos de luz vibrantes.
 * Feedback Tatil: Botoes com reacao visual ao toque (efeito de escala e brilho).
 * Links Diretos: Integracao rapida com WhatsApp, iFood e Instagram.
Estrutura de Arquivos
 * index.html: Contem a estrutura semantica, textos e os links de redirecionamento.
 * style.css: Contem todo o design visual, as animacoes de fogo (keyframes), variaveis de cor e ajustes de responsividade.
 * main.js: Gerencia a contagem de tempo de 4 segundos e desenha as particulas de brasa no fundo da tela.
Pre-requisitos (Imagens)
Para que o aplicativo funcione visualmente como planejado, voce deve colocar os seguintes arquivos de imagem (preferencialmente PNG com fundo transparente) na mesma pasta dos codigos:
 * logotipo.png (O logotipo principal da loja)
 * whatsapp.png (Icone do logo WhatsApp)
 * ifood.png (Icone do logo iFood)
 * instagram.png (Icone do logo Instagram)
Como Instalar e Usar
 * Crie uma nova pasta no seu computador.
 * Salve o codigo HTML como index.html.
 * Salve o codigo CSS como style.css.
 * Salve o codigo JavaScript como main.js.
 * Adicione as imagens listadas acima nesta mesma pasta.
 * Abra o arquivo index.html em seu navegador (Chrome, Safari, Edge) ou suba a pasta para uma hospedagem web.
Personalizacao
Para alterar os links de destino (numero do WhatsApp, link do iFood ou perfil do Instagram), abra o arquivo index.html em um editor de texto e altere o conteudo dentro das aspas em href="...".
